# FastAPI E-Commerce API

A learning-focused e-commerce REST API built with **FastAPI**.  
This project is created to practice backend development concepts such as API design, routing, validation, database models, CRUD operations, and clean project structure.

## Project Purpose

This repository is part of my learning journey with FastAPI and backend development.

The goal is to understand how a production-style API is structured step by step, including:

- REST API development
- Product management
- Request and response validation
- Database integration
- Error handling
- Clean and maintainable code structure
- Basic testing and API verification

## Tech Stack

- Python
- FastAPI
- SQLAlchemy
- Pydantic
- SQLite / Database
- Pytest
- Uvicorn

## Features

- Product create, read, update, and delete APIs
- Request body validation
- Response schema handling
- Database model structure
- API error handling
- Basic test coverage
- Modular project structure
